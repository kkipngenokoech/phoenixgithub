# PhoenixGitHub Slides

This folder contains a 5-slide HTML presentation for the Agentic AI class demo.

## Run Locally

From the project root:

```bash
cd slides
python3 -m http.server 8080
```

Open:

- `http://localhost:8080`

## Controls

- `Right Arrow` / `Space`: next slide
- `Left Arrow`: previous slide
- On-screen `Prev` / `Next` buttons also work
